Recommended install order:
1. Create/select your target MySQL or MariaDB database.
2. Import barakahfunds_v4_ultimate_mysql.sql.
3. Update includes/db.php with real database credentials.
4. Upload the full folder contents to your PHP server.
5. Ensure uploads/ and uploads/receipts/ are writable.
6. Clear OPcache / restart PHP-FPM if your server caches PHP files.
