BarakahFunds v4 - final fresh mosque edition

Included business rules:
- Only Operator can access donors, donor ledger, donation collection, and donor-linked expenses.
- Admin cannot collect donations.
- Accountant cannot collect donations.
- Only one accountant user is allowed.
- Accountant can receive operator transfers into central mosque treasury.
- Accountant can move cash into bank and see both:
  - accountant cash in hand
  - bank balance
- Events now support active/inactive switching.
- Mosque Settings now supports receipt logo upload.
- SQL file is schema-only and contains no old donor or transaction data.

Important setup note:
- Because the SQL is fresh and contains no seed data, it does not create default users automatically.
- After import, create the first admin user directly in MySQL, then log in and create:
  - one accountant
  - one or more operators

Recommended first flow:
1. Import barakahfunds_v4_fresh_management_schema.sql
2. Update includes/db.php with your database credentials
3. Create first admin user manually in users table
4. Login as admin
5. Configure Mosque Settings including receipt logo
6. Create one accountant and operator users
